# 自检索引

执行 Lint 工作流，检查 Wiki 健康状态。

## 检查项

### 1. 断链问题
检查 wikilinks 是否指向不存在的页面：
- 扫描所有 wiki 目录下的 `.md` 文件
- 提取 `[[...]]` 格式的 wikilinks
- 验证目标文件是否存在

### 2. 孤立页面
检查仅在 index.md 列出但缺少其他页面交叉引用的页面：
- 读取 `wiki/index.md` 获取所有页面列表
- 搜索每个页面在其他文件中的引用次数
- 引用次数 ≤ 1 的页面标记为孤立

### 3. 索引一致性
检查统计数据与实际文件数量是否匹配：
- 统计 `wiki/summaries/` 目录下的 `.md` 文件数量
- 统计 `wiki/concepts/` 目录下的 `.md` 文件数量
- 统计 `raw/sources/` 目录下的 `.md` 文件数量
- 与 `wiki/index.md` 中的统计数据对比

### 4. 知识矛盾（可选）
检查页面之间是否存在矛盾陈述，需要人工审核。

## 报告格式

```
## 健康检查报告 (YYYY-MM-DD)

### 🔗 断链问题
| 文件 | 断链 | 问题说明 |

### 🏝️ 孤立页面
| 页面 | 建议增加链接位置 |

### ✅ 已通过检查
- 统计核实结果
```
