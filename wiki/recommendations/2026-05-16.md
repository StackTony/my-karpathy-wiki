## 四、核心知识点整理

### 1. 中断上下半部分割

```
硬件中断触发
    ↓
上半部（Top Half）─────── 下半部（Bottom Half）
    │                         │
  关中断执行                开中断执行
  快速响应硬件              延迟处理逻辑
  ISR中断服务程序           softirq/tasklet/workqueue
  时间敏感、硬件紧密        时间不敏感、可延迟
    │                         │
    └─────→ raise_softirq() ──┘
              触发软中断
```

### 2. softirq 10种类型

| 类型 | 用途 | 说明 |
|------|------|------|
| HI_SOFTIRQ | 高优先级tasklet | 优先执行 |
| TIMER_SOFTIRQ | 定时器 | 时间回调 |
| NET_TX_SOFTIRQ | 网络发送 | 协议栈TX |
| NET_RX_SOFTIRQ | 网络接收 | 协议栈RX（高频） |
| BLOCK_SOFTIRQ | 块设备IO | 磁盘读写 |
| TASKLET_SOFTIRQ | 普通tasklet | 延迟任务 |
| SCHED_SOFTIRQ | 调度 | 负载均衡 |
| HRTIMER_SOFTIRQ | 高精度定时器 | |
| RCU_SOFTIRQ | RCU锁 | 读拷贝更新 |

### 3. ksoftirqd工作机制

```
                    软中断触发条件
                         ↓
            ┌─────────────────────────┐
            │    __do_softirq()       │
            │  循环处理pending软中断   │
            └─────────────────────────┘
                         ↓
            检查：time_before(jiffies, end)
                  && !need_resched()
                  && --max_restart
                         ↓
         ┌───────────────┴───────────────┐
         │                               │
    条件满足：继续循环              条件不满足：唤醒ksoftirqd
         │                               │
    立即执行softirq             ksoftirqd线程接管
         │                               │
         │                         低优先级执行
         │                         不影响用户进程
         └─────────────────────────────────┘
```

### 4. 中断处理完整流程

```
外设中断信号
    ↓
中断控制器（APIC）
    ↓
CPU响应中断
    ↓
保存寄存器 → 关中断
    ↓
查找中断向量表（IDT）
    ↓
执行ISR（中断服务程序）─── 上半部
    ↓
raise_softirq() 触发软中断
    ↓
开中断 → 恢复寄存器
    ↓
irq_exit() 检查pending软中断
    ↓
__do_softirq() 或 ksoftirqd ─── 下半部
    ↓
执行action回调函数
    ↓
协议栈处理/定时器回调/RCU等
```

## 五、学习路径建议

### 入门（第1天）
- 阅读 ArthurChiao 博客理解硬中断/软中断基础概念
- 区分上半部/下半部、理解为什么需要分割
- 查看 `/proc/softirqs` 和 `ps aux | grep softirq` 熟悉系统状态

### 进阶（第2天）
- 理解 softirq 10种类型及其用途
- 分析 ksoftirqd 线程触发条件（MAX_SOFTIRQ_TIME、max_restart）
- 理解 tasklet 基于 softirq 的实现（TASKLET_SOFTIRQ、HI_SOFTIRQ）

### 深入（第3天）
- 结合网卡收包场景理解中断流程完整链路
- 阅读 kernel/softirq.c 源码理解 __do_softirq() 实现
- 分析 local_bh_disable/enable 下半部禁止机制
