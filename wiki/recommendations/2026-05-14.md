## 二、已有知识总结

### 主题2：多agent编排 ✅ 已有相关知识

#### [[summaries/langchain-architecture]] 核心要点：
- LangGraph 支持图式工作流（分支、循环、并发）
- Agent 采用 **ReAct 模式**：Think → Act → Observe 循环
- AgentExecutor 决策循环 + scratchpad 中间记忆

#### [[concepts/langgraph]] 核心要点：
- StateGraph + Node + Edge + State 四大组件
- Checkpoint 持久化支持中断恢复
- 条件边实现动态路由

### 主题3：K8s云原生 ✅ 已有相关知识

#### [[summaries/kubernetes-core-concepts]] 核心要点：
- Pod/Deployment/Service/DaemonSet 核心资源
- Master-Node 架构（API Server、etcd、Scheduler、Controller）
- HPA 自动扩缩容、滚动更新回滚

#### [[summaries/containerd-runtime]] 核心要点：
- CRI 接口解耦 kubelet 与运行时
- containerd-shim 保证容器独立运行
