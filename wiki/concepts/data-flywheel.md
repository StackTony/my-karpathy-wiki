# 数据飞轮

数据飞轮（Data Flywheel）是AI原生应用的核心设计范式，强调数据的自我驱动和循环利用，通过正向反馈循环实现"越用越聪明"。

## 核心机制

```
用户交互 → 数据收集 → 评估分析 → 优化迭代 → 模型更新 → 业务应用 → 新一轮交互
    ↑                                                                        ↓
    └────────────────────────────────────────────────────────────────────────┘
```

## 反馈类型

- **显式反馈**：点赞、评分、修正答案
- **隐式反馈**：停留时长、点击位置、划走速度

## 关键特点

- **24小时闭环**：反馈数据24小时内完成微调
- **多源整合**：销售、客服、供应链数据统一
- **持续迭代**：数据积累→反馈闭环→模型优化

## AI原生5个核心零件

1. 心脏：大模型
2. 加油站：数据飞轮
3. 手脚：智能助手
4. 全感官：多模态
5. 刹车：规矩清单

## RAG应用

- **Feedback Loop**：基于用户行为优化检索策略
- **Adaptive RAG**：根据查询类型动态选择策略
- **Self RAG**：智能跳过检索
- **CRAG**：纠错式检索增强
