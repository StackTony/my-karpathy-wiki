# 红黑树

## 定义

一种**弱平衡**的二叉搜索树，通过5条颜色性质维持近似平衡（最长路径≤2倍最短路径），保证O(logn)操作效率。

## 五大性质

```
1. 节点红或黑
2. 根是黑
3. 叶子（空节点）是黑
4. 红节点的子节点是黑
5. 任一节点到叶子的路径黑节点数相同
```

## 核心特性

| 特性 | 数值 |
|------|------|
| 查找/插入/删除 | O(logn) |
| 插入旋转 | ≤1次 |
| 删除旋转 | ≤1次 |
| 最大高度 | 2×log₂(n+1) |

## 与AVL树对比

| 维度 | AVL树 | 红黑树 |
|------|-------|--------|
| 平衡严格度 | 强（差≤1） | 弱（≤2倍） |
| 旋转次数 | 多（logn级） | 少（≤1） |
| 适用场景 | 静态查找多 | 动态增删频繁 |

| 应用 | 说明 |

## 等价4阶B树

红黑树可视为4阶B树（2-3-4树）：
- 黑节点+红子节点 → 合为B树节点
- 黑节点居中，红节点居边
